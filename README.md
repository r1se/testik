# SafeMap

Map, защищённая мьютексом, с тестом конкурентного доступа.

## Структура

- `main.go` — реализация `SafeMap` и логика заполнения
- `safemap_test.go` — тест

## SafeMap

`map[int]int` под `sync.Mutex`. Публичные методы:

- `Update(key, fn)` — получить значение (создать если нет) и применить `fn`. Инкремент передаётся снаружи замыканием — отдельного метода нет.
- `Get(key)` — прочитать значение
- `AccessCount()` — число обращений к ключам
- `AddCount()` — число созданных ключей

## Тест

Ключи: от 1 до **1969** (год высадки на Луну, Apollo 11).

Запускаются 4 горутины. Горутина `id` пропускает ключи где `k % 4 == id` — каждый ключ обходят ровно 3 горутины. Порядок обхода случайный (`rand.Shuffle`).

После завершения:

| Условие | Значение |
|---|---|
| `data[k]` для любого ключа | `3` |
| `AccessCount` | `1969 × 3 = 5907` |
| `AddCount` | `1969` |
